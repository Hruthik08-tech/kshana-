# Kshana

Real time event booking and parking availability platform

## Features

### Event and Parking Discovery

1. Event discovery and booking
2. Real-time seat availability
3. Parking availability
4. Personalized recommendations
5. QR code generation and validation

---

### User Management

6. User authentication and authorization
7. Role-based access control

---

### System Features

8. Notification system
9. Payment integration

## Current Progress

1. app-requirements/ - Product requirement docs
2. database/ - Database schema and seeds
3. wireframes/ - UI wireframe assets

## Tech Stack

- Frontend: React (Vite)
- Backend: Express
- Database: MySQL
- Auth: JWT
- Real-time: WebSocket
- Workers: Python Flask
